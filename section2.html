Section 2: The Day Level Folder


                  http://gd2.mlb.com/components/game/mlb/year_YYYY/month_MM/day_DD/

Once you get to this level (on a day there is a game) you can access subfolders for each game, various .json files and various .xml files. The syntax for the subfolders are potentially the trickiest items to figure out how to properly run a routine to obtain data from them. That syntax is as follows: 

gid_YYYY_MM_DD_XXXmlb_YYYmlb_1/

Where XXX and YYY are the static 3-letter codes for each team.
Ex: Seattle = sea.
